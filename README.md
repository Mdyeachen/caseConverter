# Case Converter

A streamlined web utility for instant text transformation and analysis. Built with **React 19**, **TypeScript**, and **Tailwind CSS 4**.

## ✨ Features

* **Case Transformation**: Convert text to Sentence case, lower case, UPPER CASE, and Capitalized Case.
* **Real-time Analytics**: Live character and word counting.
* **Duplicate Detection**: Automatic identification of repeated words via a dedicated side panel.
* **One-Click Actions**: Fast copy-to-clipboard and workspace reset.
* **Modern UI**: Minimalist, responsive design optimized for focus.

## 🛠️ Tech Stack

* **React 19** - UI Framework
* **Vite 8** - Build Tool
* **Tailwind CSS 4** - Styling
* **TypeScript** - Type Safety
